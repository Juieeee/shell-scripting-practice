# Sed Practice

## Overview

This folder contains hands-on practice of the sed (Stream Editor) command in Linux.

sed is used to search, replace, delete, and print text from files. It is widely used in DevOps to automate configuration changes and process text files.

---

## Files

- employees.txt - Sample data for practice
- sed_commands.sh - Shell script demonstrating sed commands

---

## Commands Practiced

- Replace text
- Replace all occurrences
- Print a specific line
- Print a range of lines
- Delete a line
- Delete blank lines

---

## Real-world DevOps Use Cases

- Modify configuration files
- Replace IP addresses or hostnames
- Update environment variables
- Automate file editing

---

## Skills Learned

- Text manipulation
- Stream editing
- Configuration automation
- Linux command-line tools
