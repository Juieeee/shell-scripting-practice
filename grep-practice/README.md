# Grep Practice

## Overview

This folder contains hands-on practice of the grep command in Linux Shell Scripting.

grep (Global Regular Expression Print) is used to search for specific words or patterns in files. It is one of the most commonly used commands in Linux and DevOps for searching logs, configuration files, and text files.

---

## Files

- students.txt - Sample file for grep practice.
- server.log - Sample log file for log searching.
- grep_commands.sh - Shell script containing various grep commands.

---

## Commands Practiced

- Search for a word
- Ignore case (-i)
- Show line numbers (-n)
- Count matching lines (-c)
- Exclude matching lines (-v)
- Match whole words (-w)
- Search recursively (-r)
- Display filename containing the pattern (-l)

---

## Real-world DevOps Use Cases

- Search application logs for errors.
- Find specific configuration values.
- Filter log files during troubleshooting.
- Monitor server logs for failures.
- Search keywords across multiple files.
