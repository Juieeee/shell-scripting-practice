# Cron Job Practice

## Overview

This folder contains practice examples for Linux Cron Jobs.

Cron is a time-based job scheduler used to automate repetitive tasks such as backups, log cleanup, monitoring, and script execution.

---

## Files

- cron_commands.sh
- cron_examples.txt

---

## Commands Practiced

- crontab -l
- crontab -e
- crontab -r
- systemctl status cron
- systemctl restart cron

---

## Common Cron Expressions

- Every minute
- Every 5 minutes
- Every hour
- Every day at 9 AM
- Every Monday
- Every month
- Every Sunday
