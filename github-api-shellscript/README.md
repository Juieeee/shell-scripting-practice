\# GitHub API Shell Script Project



\## Overview



This project demonstrates GitHub REST API integration using Bash Shell Scripting.



The script authenticates using a GitHub Personal Access Token (PAT) and fetches repository collaborators.



\---



\## Features



\- GitHub REST API Integration

\- PAT Authentication

\- Bash Shell Scripting

\- JSON Parsing using jq

\- Linux (WSL Ubuntu)



\---



\## Tech Stack
